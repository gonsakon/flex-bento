# Flex 便當

這是一個用 Vite + Vue 3 製作的 Flexbox 學習遊戲。靈感來自 Flexbox Froggy，但關卡範圍只涵蓋第二週 Flexbox 講義。

## 範圍

第一版只使用：

- `display: flex`
- 外容器 / 內元件
- 主軸 / 交錯軸
- `flex-direction`
- `justify-content`
- `align-items`
- `flex-wrap`

第一版暫不使用：

- `order`
- `align-self`
- `flex-grow`
- `flex-shrink`
- `flex-basis`
- CSS Grid
- `gap`

## 指令

```bash
npm install
npm run dev
npm run build
```
